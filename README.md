# ML-Notebooks
Notebooks submitted for the Machine Learning course INF398 at USM (2023-2).
The course consisted of 4 workshops covering supervised learning, model evaluation, feature engineering, and domain adaptation. All workshops were completed in pairs under the team name **MCMC Metropolis**.
Two workshops were competition-based, evaluated on private Kaggle leaderboards among 11 teams:
- **Workshop 2** — Credit risk prediction: **1st place**
- **Workshop 4** — Used car price prediction (domain adaptation / transfer learning): **2nd place**

The remaining two workshops (1 and 3) were structured as guided learning exercises.
